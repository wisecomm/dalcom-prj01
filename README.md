# dalcom prj01

[shadcn/ui](https://ui.shadcn.com) 메뉴바 구성

## Features

- Retractable mini and wide sidebar
- Scrollable sidebar menu
- Sheet menu for mobile
- Grouped menu with labels
- Collapsible submenu
- Extracted menu items list

## 사용된 기술

- Next.js 15
- Shadcn/ui
- Tailwind CSS  --> 4 기반 변경 필요
- TypeScript
- Zustand
